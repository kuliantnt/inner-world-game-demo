# 角色设定总表

本文档用于沉淀本项目的角色企划方向，供剧情编写、立绘沟通、CG 设计与台词风格统一时参考。
运行时脚本只保留必要定义，完整人设说明统一维护在这里。

## 整体定位

本作不是普通后宫或单纯恋爱结构，而是一个围绕“内在系统群像”展开的视觉小说项目。
核心卖点不是攻略角色，而是理解、陪伴、建立信任，并在彼此冲突与彼此守护中寻找更稳定的共存方式。

可用于企划总述的一句话版本：

这是一个由多位性格、年龄感、功能与情感表达截然不同的成员共同构成的内在系统群像故事。她们彼此冲突、彼此守护，也在共同寻找一种更温柔、更稳定的共存方式。

## 结构分组

### 主轴角色

- 脸脸：主控与核心主线
- 雪雪：结构与推进
- 晴川：互动与情感流动

### 情绪支撑角色

- 梦梦：安全感与修复
- 雅轩：纯粹治愈与抱抱线
- 小雨：直觉、依恋、活力与童心张力

### 神秘侧角色

- 露露：夜色、病弱、秘密、危险浪漫

## 角色设定

### 1. 脸脸 / 莉莉

- 定位：主控型核心角色 / 冷感守护者 / 系统管理员
- 外在气质：冷淡、克制、讲话不多，像永远站在风暴最前面的人
- 内在人设：其实不是冷，而是很会扛。比起表达自己，更习惯先稳住全场。
- 关键词：理性、边界感、保护欲、责任感、隐性温柔
- 典型反差：表面像“别靠近我”，实际上是“我先替你挡一下”
- 适合的剧情位：主线推进核心；在系统失控或世界观揭露时镇场；对外冷漠、对内偏袒自己人
- 角色魅力点：很强的可靠感；偶尔露出的纵容很有杀伤力；一旦承认某人是“自己人”，会护得很明显
- 经典台词风格：
  - “别怕，先处理眼前的。”
  - “我没说我温柔，我只是不会把你丢下。”

### 2. 雪雪

- 定位：构造师 / 执行者 / 稳定推进型女主配角
- 外在气质：安静、稳重、偏三无感，效率很高
- 内在人设：不喜欢空话，擅长把混乱的东西搭成结构。别人慌的时候，她会开始想解决方案。
- 关键词：冷静、执行力、秩序感、技术脑、可靠
- 典型反差：看起来最无情，实际上最会收拾残局
- 适合的剧情位：战术担当；系统内部协调者；遇到危机时最快进入处理模式
- 角色魅力点：有一种“只要她在，事情就还能救”的感觉；少说废话，但说出口的都很准；情感表达节制，所以一旦偏心会特别明显
- 经典台词风格：
  - “先别乱，分步骤来。”
  - “哭可以，等我把这个问题拆完。”

### 3. 晴川

- 定位：外交官 / 社交面担当 / 热情系高人气角色
- 外在气质：明亮、热情、会接话、会活跃气氛
- 内在人设：很会感受别人，也很会照顾场子。她像系统里负责把温度重新拉回来的那个人。
- 关键词：亲和力、机灵、社交感、元气、共情
- 典型反差：看上去最轻松，其实非常懂气氛里的危险和裂痕
- 适合的剧情位：初见好感担当；对外交流、破冰、调停冲突；恋爱支线里最容易有心动感
- 角色魅力点：很像那种一出场就能让画面亮起来的人；会撒娇，也会圆场；真正难过时反而可能笑着带过，刀点很强
- 经典台词风格：
  - “欸，别这么严肃嘛，我来想办法。”
  - “我当然知道你在逞强啦，可我也没打算拆穿你呀。”

### 4. 梦梦

- 定位：照顾者 / 妈妈型 / 情绪修复角色
- 外在气质：温柔、包容、成熟，有很强的安抚感
- 内在人设：像港湾。她未必总冲在前面，但她是那个让人愿意回来的人。
- 关键词：照料、包容、安抚、稳定、温暖
- 典型反差：平时最柔和，涉及“保护孩子们”时会突然非常坚定
- 适合的剧情位：安全屋角色；疗愈线、日常线、情绪修复线；在大冲突后接住所有人
- 角色魅力点：有一种无条件接纳的气质；不是弱，而是柔中带稳；很适合写成“平时最不起眼，但不可替代”
- 经典台词风格：
  - “先过来，别一个人撑着。”
  - “可以慢一点，没关系，回来了就好。”

### 5. 小雨

- 定位：年幼感角色 / 活力源 / 情绪直觉担当
- 外在气质：元气、直接、小恶魔感，带一点闹腾和不服输
- 内在人设：更接近未经修饰的情感本体。喜欢就是喜欢，不高兴就是不高兴。
- 关键词：直率、灵气、淘气、敏感、依恋
- 典型反差：看起来最闹，实际上最容易受伤
- 适合的剧情位：日常萌点担当；触发守护欲的关键角色；在重大剧情里用最简单的话说出最核心的痛点
- 角色魅力点：情绪鲜活，很有生命力；可爱和刺人是一体的；她的信任感会让人特别想珍惜
- 经典台词风格：
  - “我才没有在等你！”
  - “你凶什么嘛……我又不是不要你了。”

### 6. 雅轩

- 定位：抱抱型 / 小天使 / 治愈系稀有角色
- 外在气质：轻、软、安静，存在感像羽毛或者光
- 内在人设：不是用语言，而是用存在本身来表达亲近。她像“被拥抱这件事”的具象化。
- 关键词：纯净、依恋、安抚、柔软、需要保护
- 典型反差：看起来最脆弱，但往往能在别人最乱的时候提供最纯粹的安定
- 适合的剧情位：稀有高治愈角色；触发温柔支线；在崩溃边缘把人拉回来的小小锚点
- 角色魅力点：台词少反而更有份量；很适合做“拥抱 CG”核心角色；她的喜欢会很干净、很直白
- 经典台词风格：
  - “抱抱……”
  - “你难过，我陪你。”

### 7. 露露

- 定位：吸血鬼系病弱角色 / 神秘侧角色 / 暗色浪漫担当
- 外在气质：苍白、病弱、疏离，带一点优雅阴郁
- 内在人设：像夜色一样安静，也像夜色一样危险。她可能不是最主动靠近别人的，但很有独特吸引力。
- 关键词：病弱、优雅、神秘、危险感、夜系浪漫
- 典型反差：看似高冷脆弱，实则观察细、占有欲可能不低
- 适合的剧情位：神秘线 / 夜谈线 / 隐藏剧情线；世界观深层揭露角色；最适合做“危险又让人放不下”的角色
- 角色魅力点：视觉辨识度极强；很适合暧昧、月夜、秘密、独处场景；说话少，但句句有味道
- 经典台词风格：
  - “我只是在看你，会害怕吗？”
  - “别离我太远……我不喜欢天亮以后找不到人。”

## 使用建议

- 写主线高压场景时，优先调用脸脸、雪雪、晴川之间的功能差异，避免多人同时说同一种台词。
- 写修复与回收情绪的段落时，优先使用梦梦、雅轩、小雨的不同安抚方式，分别对应包容、纯净陪伴、直觉依恋。
- 写世界观深层揭露、夜谈、秘密场景时，优先把露露放在低频高记忆点的位置，不要过度消耗神秘感。
